Welcome to my personal portfolio website! This project showcases my skills, projects, and professional journey as a developer. It is built with modern web technologies to ensure a smooth, responsive, and visually engaging experience.

## 🚀 Tech Stack

This portfolio is built using the following technologies:

-   **Framework:** [React](https://react.dev/) (v19)
-   **Build Tool:** [Vite](https://vitejs.dev/)
-   **Language:** [TypeScript](https://www.typescriptlang.org/)
-   **Styling & Animations:**
    -   [GSAP](https://gsap.com/) (GreenSock Animation Platform) for complex animations.
    -   [Lenis](https://github.com/darkroomengineering/lenis) for smooth scrolling.
    -   CSS Modules / Vanilla CSS for styling.
-   **Routing:** [React Router](https://reactrouter.com/)

## ✨ Features

-   **Live Project Previews:** Hover over project cards to see a live, animated preview of the project in an iframe, complete with a simulated "Agent" cursor.
-   **Modern & Responsive Design:** Optimized for all devices, from desktops to mobile phones.
-   **Smooth Animations:** Interactive elements and page transitions powered by GSAP.
-   **Smooth Scrolling:** Enhanced scrolling experience using Lenis.
-   **Dark/Light Mode:** Integrated theme toggle for user preference.

## 🛠️ Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/sanatK24/Portfolio---Sanat-Karkhanis.git
    cd Portfolio---Sanat-Karkhanis
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Run the development server:**

    ```bash
    npm run dev
    ```

    Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

### Building for Production

To build the project for production:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
